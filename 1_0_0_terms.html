The DevLab Multi Hub Shield integrates with various ecosystems and standards:

  • QWIIC/STEMMA QT: SparkFun and Adafruit connector standard for I2C devices

  • Gravity: DFRobot sensor connector standard

  • Arduino Libraries: Various open-source libraries for displays, sensors, and peripherals

  • MicroPython: Python implementation for microcontrollers
